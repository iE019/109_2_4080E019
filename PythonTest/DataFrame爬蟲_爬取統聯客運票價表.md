```
pandas 套件，是一種常用的 Data 分析處理工具,
類似 Excel，能夠將儲存的 Data 進行統計、搜尋或修改等操作.

主要有兩種 Data structure，分別是 Series 及 DataFrame

DataFrame 主要用來處理雙維度的資料
具有 row(橫列) 與 column(直欄)的 Table(表格)式資料集
所以經常應用於讀取 CSV檔案、Web table (網頁表格) 或 DataBase(資料庫)等，來進行其中的 Data 分析或處理

lxml 套件是用來幫助 Pandas 套件在讀取網頁 table(表格)Data 時，解析 HTML 原始碼
```

## read_html() method 說明
```
read_html() method，
只要傳入網址，就能夠讀取網頁中的 <table> 表格標籤，
並且回傳一個 Pandas DataFrame 物件，
<tr>標籤 是 Pandas DataFrame 資料結構的 row(橫列)，
<td>標籤 是 column(直欄)。

read_html() method 會將參數內的放入的網址的網頁中的所有表格組成一個 List(串列)
```

## 實作1_爬取所有表格
```python
import pandas as pd
# 匯入 pandas 套件 並命名為 pd
 
bus_table = pd.read_html("https://www.ubus.com.tw/Booking/FareInquiry")
# 參數 is 統聯客運票價表
# 讀取統聯客運的所有 table 並組成 a List

print(bus_table)
# 以 DataFrame 的格式列印出爬取結果
```
