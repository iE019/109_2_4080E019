# 字體

*斜體字 1*  

_斜體字 2_

## 說明

```
斜體字

方法一: 前後加入 1個 星號(星號 與 字 之間不能有空格)

方法二: 前後加入 1個 底線(底線 與 字 之間不能有空格)
```

**粗體字**  

__粗體字__

## 說明

```
粗體字

方法一: 前後加入 2個 星號(星號 與 字 之間不能有空格)

方法二: 前後加入 2個 底線(底線 與 字 之間不能有空格)
```

***粗體 + 斜體***  

___粗體 + 斜體___

## 說明

```
粗體 + 斜體

方法一: 前後加入 3個 星號(星號 與 字 之間不能有空格)

方法二: 前後加入 3個 底線(底線 與 字 之間不能有空格)
```
