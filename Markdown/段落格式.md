# 字體

*斜體字 1*
_斜體字 2_

## 說明

```
斜體字

方法一: 前後加入 1個 星號

方法二: 前後加入 1個 底線
```

** 粗體字 **
__ 粗體字  __

## 說明

```
粗體字

方法一: 前後加入 2個 星號

方法二: 前後加入 2個 底線
```

*** 粗體 + 斜體 ***
___ 粗體 + 斜體 ___

## 說明

```
粗體 + 斜體

方法一: 前後加入 3個 星號

方法二: 前後加入 3個 底線
```
